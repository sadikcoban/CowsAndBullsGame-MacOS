# CowsAndBullsGame-MacOS

A basic MacOS game application written in Swift to practice some of the futures of MacOS app development including;

- NSTableView
- NSTextField
- NSButton 
- NSTableViewCell
- NSTableColumn
- GamePlayKit
- enumerated
- Set
- CharacterSet
- insertion on tableview
- NSAlert
